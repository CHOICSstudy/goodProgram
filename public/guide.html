📚 goodProgram 사용법

팀 인강 계정 공유 시간 관리 — 동시 접속 충돌 없이 인강 듣기

1. 들어가기

  1. 사이트 접속 → 팀 공용 비밀번호 입력 (팀에서 공유받은 비밀번호)
  2. 자기 이름 선택 — 체크인·예약이 이 이름으로 기록됩니다
  3. 이름을 잘못 골랐으면 오른쪽 위 변경을 누르세요
🔒 비밀번호를 5회 연속 틀리면 1분간 입력이 잠깁니다.

2. 인강 목록 — 지금 들을 수 있나?

각 인강 카드에 묶인 계정과 현재 상태가 표시됩니다.

표시의미
🟢 사용 가능지금 아무도 안 쓰는 중 — 바로 사용 가능
🔴 OO 사용 중 (~18:00)OO님이 사용 중, 18:00에 자동 종료 예정
다음 예약: OO 14:00~15:00오늘 그 계정에 잡혀있는 예약 (참고용)
  • 사용 시작(체크인): 종료 예정 시간을 고르고 클릭 — 30분~6시간 선택 또는 직접 입력(최대 10시간)
  • 미정으로 시작하면 2시간 후 자동으로 해제됩니다
  • 사용 종료(체크아웃): 다 들었으면 꼭 눌러주세요 — 다른 팀원이 바로 쓸 수 있게
  • 위쪽 카테고리 버튼과 검색창으로 인강을 찾을 수 있습니다

3. 계정 확인하기 — 어떤 계정으로 로그인?

  1. 인강 카드의 계정 확인하기 클릭
  2. 모달에서 로그인할 사이트와 아이디/비밀번호 확인 — 각각 복사 버튼 제공
  3. 비밀번호는 기본 가려져 있고 보기를 눌러 확인
  4. 모달 안에서도 바로 체크인/체크아웃 가능
⚠️ 계정 정보는 팀 내부용입니다. 외부에 공유하지 마세요.

4. 시간표 — 미리 예약하기

  1. 왼쪽에서 계정 선택 — 계정 아래에 묶인 인강 목록이 보입니다 (▾/▸로 접기/펼치기)
  2. 주간 표(월~일, 09:00~24:00, 30분 단위)에서 원하는 시간을 잡으세요
  3. 한 칸 클릭 = 30분 예약, 아래로 드래그 = 원하는 만큼 한 번에 예약
  4. 다이얼로그에서 들을 인강을 고르고(선택) 예약 클릭
  • 초록 칸 = 예약됨 (예약자 이름 표시)
  • 파란 칸 = 드래그로 선택 중인 범위
  • 내 예약은 칸을 클릭하면 취소할 수 있습니다 (남의 예약은 클릭해도 반응 없음)
  • 같은 계정·같은 시간에 겹치는 예약은 자동으로 막힙니다

5. 관리 — 계정/인강 추가·수정

상단 관리 탭에서 계정과 인강을 추가·수정·삭제할 수 있습니다. 새 인강을 결제했거나 계정 정보가 바뀌면 여기서 업데이트하세요.

⚠️ 계정 삭제 시 그 계정에 묶인 인강과 예약도 함께 삭제됩니다.

6. 이용 규칙 (팀 약속)

  • 인강을 듣기 시작할 때 꼭 체크인, 끝나면 꼭 체크아웃
  • 체크아웃을 깜빡해도 예정 시간(미정이면 2시간)이 지나면 자동 해제되지만, 그동안 다른 팀원이 못 쓰니 주의
  • 오래 들을 계획이면 시간표에 미리 예약해서 팀원과 겹치지 않게
  • 화면은 몇 초마다 자동 갱신되므로 새로고침 없이도 최신 상태가 보입니다
